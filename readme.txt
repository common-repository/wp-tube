== Description ==

The WP-Tube for YouTube plugin can automaticaly import your Youtube videos including their description, keywords and attributes right into your blog stream. It uses the Youtube, GData API to get the videos and you can customize it specifying which Youtube username to use and how often to 'ping' the Youtube API to check for new videos as well as setting the size and color of the Youtube player.

Comming soon...
Custom template for layout as well as support for custom YouTube players such as 'Youtube with style'
